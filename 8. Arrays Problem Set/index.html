printReverse()

Write a function called printReverse() that takes an array as an argument and prints out the elements in the array in reverse order - but doesn't actually reverse the array itself 

      printReverse([1,2,3,4]);
      //4
      //3 
      //2 
      //1 
      
      printReverse(["a", "b", "c"]);
      //"c"

isUniform()

Write a function called isUniform() which takes an array as an argument and returns true is all elements in the array are identical. 

      isUniform([1, 1, 1, 1]);     //true 
      isUniform([1, 2, 1, 1]);     //false
      isUniform(["a", "b", "p"]);  //false
      isUniform(["b", "b", "b"]);  //true

sumArray()

Write a function called sumArray() that accepts an array of numbers and returns the sum of all the numbers in the array. 

      sumArray([1,2,3]);      //6 
      sumArray([10,3,10,4]);  //27 
      sumArray([-5,100]);     //95
